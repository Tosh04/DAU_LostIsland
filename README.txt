Current state :
Après une tentative d'installation compliqué et le manque de documentation sur godot en mode C++, j'ai décidé de faire le projet sur unreal engine. Ainsi le projet sous unreal engine 4 vient d'être créé. Le projet contient ainsi une scène basique, une caméra en vue top-down et un personnage, qui se déplace par défaut en utilisant la souris (en cliquant à l'endroit où l'on désire se rendre).

What has been implemented :
Unreal engine 4, un template de jeu top-down

How to run the code :
Avoir la version 4.27.2 de Unreal. Cliquer sur le bouton play dans unreal
